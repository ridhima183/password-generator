## 🌈 What is this?

A **simple, colorful, and beginner-friendly** password generator built using plain **HTML, CSS, and JavaScript**! No frameworks, no fuss — just clean vanilla code that creates strong random passwords based on whatever options you pick. 🎨🔑

<br>

## 🎀 Features

| 🚀 Feature | 💬 Description |
|:---:|:---|
| 🎲 | Generate completely random passwords |
| 📏 | Adjustable password length via slider |
| 🔠 | Toggle uppercase letters on/off |
| 🔢 | Toggle numbers on/off |
| ✨ | Toggle special symbols on/off |
| 📋 | One-click copy to clipboard |
| 📱 | Fully responsive, mobile-friendly UI |

## 🚦 How to Run

```bash
# 1️⃣ Clone or download this repo
git clone https://github.com/your-username/password-generator.git

# 2️⃣ Move into the project folder
cd password-generator

# 3️⃣ Open index.html in your browser 🎉
```

Then just:
1. 🎚️ Slide to pick your password length
2. ✅ Check the boxes for uppercase, numbers, or symbols
3. 🔘 Hit **Generate Password**
4. 📋 Hit **Copy** and paste it wherever you need!
