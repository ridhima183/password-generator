# Password Generator

A simple and customizable password generator built with **HTML**, **CSS**, and **JavaScript**.  
It allows users to create strong random passwords based on selected options like length, uppercase letters, numbers, and symbols.

## Features

- Generate random passwords.
- Choose password length.
- Include uppercase letters.
- Include numbers.
- Include symbols.
- Copy password to clipboard.
- Responsive and beginner-friendly UI.

## Tech Stack

- HTML
- CSS
- JavaScript

## How to Run

1. Clone or download this project.
2. Open the folder in VS Code or any code editor.
3. Open `index.html` in your browser.
4. Use the slider and checkboxes to set your preferences.
5. Click **Generate Password**.
6. Click **Copy** to copy the password.
